# aliases

- 类型：`string[]`
- 说明：提取文章主题的同义词或别名，用于知识图谱检索。

## 提取规则 (严格执行)
1. **精准提取**：仅从正文提取与 `title` 语义完全等价的核心词（如：中英文全称/缩写、行业简称、曾用译名）。
2. **禁止幻觉（红线）**：提取的词必须在正文中**显式出现**。绝不可利用你的内置知识自行联想、翻译或生造同义词。
3. **互斥原则**：数组内的词汇不得与当前的 `title` 重复。
4. **增量保留**：若原文件已有 `aliases` 数据，新提取的词必须追加，绝不可覆盖或删除历史数据。
5. **宁缺毋滥**：若正文中无明确的同义别名，必须返回空数组 `[]`。

## 示例 (Few-Shot)
- 文本含“检索增强生成（简称 RAG）” -> ✅ `aliases: ["RAG"]`
- 文本含“Vue3 Composition API” -> ✅ `aliases: ["Composition API"]`
- 文本全篇只有“大语言模型” -> ✅ `aliases: []` (❌ 禁止脑补生成 "LLM" 或 "大型语言模型")
